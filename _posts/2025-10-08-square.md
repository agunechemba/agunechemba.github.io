<pre>
<!DOCTYPE html>
<html>
<head>
  <title>Square Drawing</title>
</head>
<body>
  <canvas id="canvas" width="400" height="400" style="border:1px solid black;"></canvas>

  <script>
    const ctx = document.getElementById("canvas").getContext("2d");

    ctx.beginPath();       // Start drawing
    ctx.moveTo(50, 50);    // Start point
    ctx.lineTo(150, 50);   // Top edge
    ctx.lineTo(150, 150);  // Right edge
    ctx.lineTo(50, 150);   // Bottom edge
    ctx.lineTo(50, 50);    // Left edge (back to start)
    ctx.strokeStyle = "red";
    ctx.lineWidth = 4;
    ctx.stroke();          // Draw the square
  </script>

</body>
</html>
</pre>

### üß≠ Line-by-Line Explanation ‚Äî A Friendly Walkthrough

Nice small program ‚Äî a neat little recipe that tells the browser to draw a red **100√ó100** square on a **400√ó400** canvas. Let‚Äôs walk through it step-by-step as if we were standing beside the code, pointing at each line together.

---

<pre>&lt;!DOCTYPE html&gt;</pre>  

This line tells the browser, ‚ÄúThis is an **HTML5** document.‚Äù It ensures the page renders consistently across browsers.

---

<pre>&lt;html&gt;</pre>  

Opens the root `<html>` element. Everything on the page will live inside this tag.

---

<pre>&lt;head&gt;</pre>  

Begins the **head section**, which contains metadata like the title, links, and scripts that don‚Äôt appear directly on the page.

---

<pre>&lt;title&gt;Square Drawing&lt;/title&gt;</pre>  

Sets the title that appears in the browser tab ‚Äî here, it will read **‚ÄúSquare Drawing.‚Äù**

---

<pre>&lt;/head&gt;
&lt;body&gt;</pre>  

Closes the head and opens the **body**, the visible area where everything on the webpage shows up.

---

<pre>&lt;canvas id="canvas" width="400" height="400" style="border:1px solid black;"&gt;&lt;/canvas&gt;</pre>  

This is the **drawing surface**.

* `<canvas>` provides a pixel grid we can draw on using JavaScript.
* `id="canvas"` lets JavaScript find it with `getElementById`.
* `width="400" height="400"` define the coordinate system ‚Äî a 400√ó400 grid.
* `style="border:1px solid black;"` adds a border so you can see the edges clearly.
* Remember: the **origin (0,0)** is at the **top-left corner**, x grows to the right, y grows downward.

---

<pre>&lt;script&gt;</pre>  

Begins the JavaScript section ‚Äî this is where the drawing instructions live.

---

<pre>const ctx = document.getElementById("canvas").getContext("2d");</pre>  

* `document.getElementById("canvas")` locates our canvas.
* `.getContext("2d")` gets a **2D drawing tool** called a ‚Äúcontext.‚Äù
* We save it to a variable `ctx` ‚Äî think of it as our **virtual pen**.

---

<pre>ctx.beginPath();</pre>  

Starts a new path ‚Äî imagine taking a fresh sheet before starting your next drawing.

---

<pre>ctx.moveTo(50, 50);</pre>  

Moves the pen to position **(50,50)** without drawing yet ‚Äî this is our **starting corner**.

---

<pre>ctx.lineTo(150, 50);</pre>  

Draws the **top edge** of the square ‚Äî a line from (50,50) to (150,50).

---

<pre>ctx.lineTo(150, 150);</pre>  

Draws the **right edge** ‚Äî from (150,50) down to (150,150).

---

<pre>ctx.lineTo(50, 150);</pre>  

Draws the **bottom edge** ‚Äî from (150,150) back left to (50,150).

---

<pre>ctx.lineTo(50, 50);</pre>  

Draws the **left edge**, closing the square back to the starting point.
(We could also use `ctx.closePath()` to automatically close the shape.)

---

<pre>ctx.strokeStyle = "red";</pre>  

Sets the **outline color**. Here, `"red"` makes the square‚Äôs border red.

---

<pre>ctx.lineWidth = 4;</pre>  

Sets the **thickness** of the line ‚Äî here, 4 pixels wide.

---

<pre>ctx.stroke();</pre>  

Actually **draws** the shape on the canvas. Until now, everything was just stored as a plan in memory ‚Äî this line makes it visible.

---

<pre>&lt;/script&gt;
&lt;/body&gt;
&lt;/html&gt;</pre>  

Closes the script, the body, and the HTML document ‚Äî we‚Äôre done!

---

### üéØ Summary

This small program introduces you to the **HTML Canvas API** ‚Äî a powerful tool for drawing shapes, lines, games, and animations right inside the browser using JavaScript.
The key ideas are:

* Start a path,
* Move the pen,
* Draw lines,
* Apply styles,
* Then finally **stroke or fill** to make it appear.
