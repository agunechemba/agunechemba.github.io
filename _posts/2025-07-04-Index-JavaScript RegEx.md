**Regular Expressions (Regex)** are **powerful search patterns** used to find, match, and manipulate text.

Think of Regex as a *smart filter* ‚Äî it can detect and extract information based on **patterns**, not just specific words.

For example:

* You can find all email addresses in a document.
* You can check if a phone number has the right format.
* You can replace all extra spaces, punctuation, or unwanted characters in a paragraph.

Regex works across almost every modern programming language ‚Äî **Python**, **JavaScript**, **Java**, **C#**, **PHP**, and many others ‚Äî because pattern matching is a universal need.
